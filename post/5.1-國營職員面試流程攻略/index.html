5.1-國營職員面試流程攻略

December 24, 2020
國考攻略手冊

106年有幸進入國營面試，因為有點時間了，現在只能盡力回想當初的面試情況，分享給即將進入面試的你們做參考，參考喔!詳細的還是要以該年度簡章及收到紙本複試通知單為主!

106年國營面試流程(無體測)

一、事前準備

  1. 確認複試時間及地點
    1.1 報到時間:2/16 17:30~17:40
    1.2 報到地點:台電公司台北市區營業處，業務大樓12樓大禮堂

  2. 確認複試需攜帶文件
    2.1 複試通知單
    2.2 成績單
    2.3 6個月內脫帽1吋照片乙張(背面先寫好姓名及入場證號碼)
    2.4 畢業證書正本及影本
    2.5 簡歷表4份(一份正本，三份影本)
    2.6 身份證正本及影本

  3. 服裝部分:畢竟面試官都是經濟部及國營事業長官，穿正式服裝會讓人感覺到「看重」和「有禮貌」，建議還是要著正式服裝。（如果有體試的，可以帶一套方便活動的服裝去換喔！）
    3.1 建議男生西裝，女生套裝
    心得:
    回想當初我穿白襯衫配藍色牛仔褲(上衣有紮進去)，結果一到現場有點小慌張，整個會場只有少數2~3個不是穿著西裝，超級突兀的XD不知道穿著正式服裝是否有加分，但很多時候都是相比下來，人家有穿你沒穿，保險起見還是著正式服裝。

  4. 其他:若是面試時間剛好卡在吃飯時間，外加要填寫、繳交資料及不確定面試是在前面還是後面，若不幸在後面可能會待完全程(約4小時)，建議可以準備一些小點心，免得餓昏頭影響到面試。
    溫馨小提醒:
    若是有吃東西，面試前記得照一下鏡子，確保儀態。面試時間通常落在1~2月左右，記得攜帶保暖衣物喔!

二、報到

  1. 建議提早15~30分鐘到，早到會被警衛帶到會客室等。如果是帶西裝來換的，可以利用等待時換。
  2. 時間到會有人來帶你們搭電梯上去。
  3. 出電梯會拿給你識別證、(紅、黃、藍)其一個顏色的緞帶及按照複試編號就座。

三、填寫及繳交資料

  1. 桌上會有複試紀錄表、志願表、專A(大概10題)及短文寫作試紙，就座後就開始填寫上述四份資料。作文是跟筆試的題目一樣，大概寫個5~7行即可；專A跟作文聽說是用來防槍手的。
  2. 會依緞帶顏色分批交書面資料及收作文。
    心得:
    如果是被排到後面面試的，在等待的過程中，不妨可以跟附近的聊聊天，舒解緊張的情緒及探聽情報XD
    後面面試的有一個好處，先口試完的會回到座位，這時不妨可以跟他們請教考官都問了哪些問題或是其他資訊~

四、面試開始

  1. 每次會叫18個人去拍大頭照，拍完後去面試場地外坐著等。
  2. 面試時2人一組，共18分鐘。每組會有3~4個面試官，面試官組成為油、電、糖、水及經濟部長官。
  3. 面試內容寫在下一章節裡喔。
  4. 場地內使用屏風隔開各組。

五、面試結束

  1. 領回證件及資料。
